Портфолио

Проекты мини

  • Таймер обратного отсчёта

    Простой таймер обратного отсчета с днями, часами, минутами и секундами. Ведет обратный отсчет до Нового Года.

  • Генератор цитат

    При загрузке страницы приложения отображается рандомная цитата, есть возможность обновить цитату в ручную.

  • Список задач

    Приложение позволяет добавлять задачи в список, помечать их, как выполдненные, и удалять.

  • Аудио плеер

    Музыкальный плеер, позволяющий по очереди проигрывать музыкальные треки или перелистывать их кликами по кнопкам.

  • CSS Bayan

    Aккордеон реализованный только с помощью HTML и CSS.
